Установка

Установка релизной версии

Скопируйте файл FS17_CoursePlay5.zip в папку модов. Вот и всё, готово!

Папка модов может иметь такое расположение:

  • Windows (7 и выше):
  • C:\Users\[USER NAME]\My Documents\MyGames\FarmingSimulator2017\mods
  • Mac OS X (Download/DVD/Steam-Version):
  • /Users/[USER NAME]/Library/Application Support/FarmingSimulator2017
  • Mac OS X (X 10.7 и выше):
  • /Users/[USER NAME]/Library/Containers/com.focus-home.farmingsim2017/Data/Library/Application Support/FarmingSimulator2017

Установка версии разработчиков

Сначала убедитесь, что у вас установлен последний патч: Farming Simulator 17 Update

Получите последнюю версию разработчиков: Developer Version

Распакуйте загруженный архив в выбранную папку.
Внутри вы найдете папку с названием courseplay-master.
Выделите все Файлы и Папки внутри папки courseplay-master и заархивируйте выбранное в zip-архив, с названием FS17_CoursePlay5.zip.
Затем выполните процесс установки так же как для релизной версии (см. выше).
